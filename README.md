這是一個基於經典 Flappy Bird 改編的 Web 小遊戲，我們加入了「重力反轉」與「時間膨脹」等量子物理概念，並採用 Cyberpunk 霓虹風格設計。

[🎮 點此試玩 (Demo Link)](https://dusybog.github.io/week18_group2/)

## ✨ 特色與創新玩法

除了傳統的飛行閃避障礙物，本遊戲包含以下創新機制：

1.  **🟣 重力反轉 (Gravity Invert)**:
    - 吃到隨機出現的「紫色晶片」後，重力場會瞬間顛倒。
    - **[Risk & Reward]**：雖然操作難度會瞬間提升（變成點擊=向下衝），但成功獲得晶片將直接 **獎勵 5 分**，鼓勵玩家冒險嘗試。
    
2.  **⏳ 時間膨脹 (Time Dilation)**:
    - 當分數達到 10 的倍數時，觸發子彈時間。
    - 遊戲速度減緩 50%，給你更多反應時間來處理複雜地形。

## 🛠 技術堆疊

* **Frontend**: Native HTML5 Canvas, CSS3 (Flexbox/RWD), Vanilla JavaScript (ES6+).
* **Deployment**: GitHub Pages.

## 🚀 安裝與執行

1.  Clone 本專案：
    ```bash
    git clone https://github.com/dusybog/week18_group2.git
    ```
2.  直接用瀏覽器打開 `index.html` 即可遊玩。
